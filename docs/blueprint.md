# **App Name**: IssueTrackr

## Core Features:

- User Authentication: Secure user login and registration to manage access to the system.
- Issue Reporting: Allow users to submit new IT issues with detailed descriptions, attachments, and urgency levels.
- Issue Tracking: Track the status and history of each issue, from submission to resolution, with audit trails.
- Dashboard Overview: Provide a dashboard with a summary of pending and resolved issues.
- Smart Summarization: Use AI to provide summaries of current tickets.

## Style Guidelines:

- Primary color: Soft blue (#72BCD4) to evoke trust and stability.
- Background color: Light gray (#F0F4F7), a desaturated version of the primary.
- Accent color: Orange (#FF9800) for important actions and notifications.
- Headline font: 'Space Grotesk', a sans-serif font.
- Body font: 'Inter', a sans-serif font.
- Simple, consistent icons for issue types and status indicators.
- Clean, organized layout for the dashboard and issue details.
- Subtle animations for loading states and transitions.
